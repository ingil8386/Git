HTML 테이블 태그 실습

테이블 태그는 표를 만들 때 사용합니다.

1. 기본 테이블

이름 나이 지역
김철수 20 서울
이영희 22 부산

table은 표 전체를 의미합니다.
tr은 행, td는 칸을 의미합니다.

2. 제목 셀이 있는 테이블

번호 상품명 가격
1 아메리카노 3000원
2 카페라떼 4000원
3 바닐라라떼 4500원

th는 제목 칸입니다.
td보다 글자가 굵고 가운데 정렬됩니다.

3. thead, tbody, tfoot 사용

과목 점수 등급
HTML 90 A
CSS 85 B
JavaScript 70 C
평균 81.6 B
아이디 이름 생년월일 나이 관리

thead는 표의 머리 부분입니다.
tbody는 표의 본문 부분입니다.
tfoot은 표의 마지막 요약 부분입니다.

caption 태그는 표의 제목을 나타냅니다.

rowspan은 세로 방향으로 칸을 합칠 때 사용합니다.

7. colspan과 rowspan 같이 사용

rowspan은 세로로 합치기, colspan은 가로로 합치기입니다.

9. 테이블 태그 정리

태그 의미
<table> 표 전체
<tr> 표의 한 행
<td> 일반 칸
<th> 제목 칸
<caption> 표 제목
<thead> 표 머리 부분
<tbody> 표 본문 부분
<tfoot> 표 마지막 요약 부분
colspan 가로 칸 합치기
rowspan 세로 칸 합치기
