HTML 시멘틱 태그란?

작성일: 2026년 5월 11일

시멘틱 태그는 태그 이름만 보고도 해당 영역의 의미를 알 수 있게 해주는 태그입니다. 예를 들어 header는 머리말, nav는 메뉴, footer는 하단 영역을 의미합니다.

카테고리: HTML 기초

시멘틱 태그를 사용하는 이유

작성일: 2026년 5월 12일

시멘틱 태그를 사용하면 코드의 구조가 명확해지고, 검색 엔진과 브라우저가 문서의 의미를 더 잘 이해할 수 있습니다.
