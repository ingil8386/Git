HTML 텍스트 태그와 목록 태그 실습

이번 실습에서는 HTML에서 자주 사용하는 텍스트 태그와 목록 태그를 연습합니다.

1. 제목 태그

제목 태그는 문서의 제목이나 소제목을 나타낼 때 사용합니다. h1부터 h6까지 있으며, 숫자가 작을수록 더 큰 제목입니다.

h1 가장 큰 제목

h2 두 번째 제목

h3 세 번째 제목

h4 네 번째 제목

h5 다섯 번째 제목
h6 가장 작은 제목

2. 문단 태그

p 태그는 문단을 만들 때 사용합니다.

p 태그를 여러 번 사용하면 이렇게 문단이 나누어집니다.

HTML 코드에서 엔터를 쳐도 브라우저 화면에서는 줄바꿈이 바로 적용되지 않을 수 있습니다.

줄을 바꾸고 싶으면
br 태그를 사용합니다.

3. 강조 태그

강조 태그는 특정 단어나 문장을 눈에 띄게 표현할 때 사용합니다.

이 문장은 b 태그로 굵게 표시했습니다.

이 문장은 strong 태그로 중요한 내용을 표시했습니다.

이 문장은 i 태그로 기울임 표시했습니다.

이 문장은 em 태그로 강조 표시했습니다.

중요한 부분은 mark 태그로 형광펜 표시를 할 수 있습니다.

작은 설명은 small 태그로 작게 표시할 수 있습니다.

삭제된 내용은 del 태그로 삭제선을 표시합니다.

추가된 내용은 ins 태그로 밑줄을 표시합니다.

4. 아래첨자와 위첨자

sub 태그는 아래첨자, sup 태그는 위첨자를 표현할 때 사용합니다.

물의 화학식은 H2O 입니다.

이산화탄소는 CO2 입니다.

2의 제곱은 22 입니다.

3의 제곱은 32 입니다.

5. 인용 태그

q 태그는 짧은 인용문을 표현할 때 사용합니다.

선생님이 말씀하셨습니다. HTML은 구조가 중요하다.

blockquote 태그는 긴 인용문을 표현할 때 사용합니다.

HTML은 웹 페이지의 구조를 만드는 언어입니다. 제목, 문단, 목록, 링크, 이미지 등을 태그로 표현합니다.

6. 코드 표시 태그

code 태그는 코드처럼 보이게 표시할 때 사용합니다.

문단 태그는 <p> 입니다.

줄바꿈 태그는 <br> 입니다.

제목 태그는 <h1>부터 <h6>까지 있습니다.

pre 태그는 줄바꿈과 공백을 작성한 그대로 보여줍니다. 

HTML 기본 구조

html
 ├─ head : 문서 정보
 └─ body : 화면에 보이는 내용

7. 링크 태그

a 태그는 다른 페이지로 이동하는 링크를 만들 때 사용합니다.

네이버로 이동

구글 새 탭으로 이동

target="_blank"를 사용하면 새 창 또는 새 탭으로 열립니다.

8. 순서 없는 목록 태그

순서가 중요하지 않은 목록은 ul 태그를 사용합니다. 각각의 항목은 li 태그로 작성합니다.

좋아하는 음식

  • 김치찌개
  • 제육볶음
  • 돈까스
  • 라면

필요한 준비물

  • 노트북
  • 충전기
  • 마우스
  • 필기구

9. 순서 있는 목록 태그

순서가 중요한 목록은 ol 태그를 사용합니다. 브라우저에서 숫자가 자동으로 붙습니다.

11. 설명 목록 태그

용어와 설명을 함께 정리할 때는 dl, dt, dd 태그를 사용합니다.

HTML

CSS
웹 페이지의 디자인을 담당하는 언어입니다.
JavaScript
웹 페이지에 동작을 추가하는 언어입니다.
ul
순서 없는 목록을 만들 때 사용하는 태그입니다.
ol

li

12. 텍스트 태그 종합 실습

저는 오늘 HTML 텍스트 태그와 목록 태그를 공부했습니다.

처음에는 head 안에 화면 내용을 작성했지만, 이제는 body 안에 화면 내용을 작성해야 한다는 것을 알았습니다.

특히 ul, ol, li 태그는 목록을 만들 때 자주 사용됩니다.

HTML에서 중요한 말은 화면에 보이는 내용은 body 안에 작성한다입니다.

14. 전체 정리

HTML의 텍스트 태그는 글자를 제목, 문단, 강조, 인용, 코드 형태로 표현할 때 사용합니다.

목록 태그는 여러 개의 항목을 보기 좋게 정리할 때 사용합니다.

<h1> ~ <h6>
제목을 작성할 때 사용합니다.
<p>
문단을 작성할 때 사용합니다.
<br>
줄바꿈을 할 때 사용합니다.
<hr>
가로선을 만들 때 사용합니다.
<strong>
중요한 내용을 표시할 때 사용합니다.
<mark>
형광펜처럼 강조할 때 사용합니다.
<ul>
순서 없는 목록을 만들 때 사용합니다.
<ol>
순서 있는 목록을 만들 때 사용합니다.
<li>
목록의 항목을 작성할 때 사용합니다.
<dl>, <dt>, <dd>
용어와 설명을 정리할 때 사용합니다.
