HTML 기본 태그 실습

이 페이지는 HTML의 기본 텍스트 태그를 연습하기 위한 예제입니다.

1. 제목 태그

h1 제목 태그입니다.

h2 제목 태그입니다.

h3 제목 태그입니다.

h4 제목 태그입니다.

h5 제목 태그입니다.
h6 제목 태그입니다.

제목 태그는 h1부터 h6까지 있습니다. 숫자가 작을수록 글자가 크고, 숫자가 클수록 글자가 작아집니다.

2. 문단 태그

p 태그는 문단을 만들 때 사용합니다.

p 태그를 여러 번 사용하면 문단이 나누어집니다.

HTML에서는 코드에서 줄바꿈을 해도 브라우저 화면에서는 자동으로 줄바꿈되지 않을 수 있습니다.

줄을 바꾸고 싶으면
br 태그를 사용하면 됩니다.

3. 링크 태그

아래는 링크 태그 예제입니다.

네이버로 이동

구글 새 창으로 이동

a 태그는 다른 페이지로 이동할 때 사용합니다. target="_blank"를 사용하면 새 창 또는 새 탭에서 열립니다.

4. 강조 태그

이 문장은 굵게 표시됩니다.

이 문장은 중요한 내용입니다.

이 문장은 기울임 표시됩니다.

이 문장은 강조된 내용입니다.

이 문장은 형광펜 표시됩니다.

이 문장은 작은 글씨입니다.

이 문장은 삭제된 내용입니다.

이 문장은 추가된 내용입니다.

5. 위첨자와 아래첨자

물의 화학식은 H2O 입니다.

2의 제곱은 22 입니다.

6. 인용 태그

짧은 인용문은 q 태그를 사용합니다.

HTML에서 구조가 중요하다라고 말할 수 있습니다.

blockquote 태그는 긴 인용문을 작성할 때 사용합니다. 보통 들여쓰기 형태로 화면에 표시됩니다.

7. 코드 표시 태그

HTML에서 문단을 만들 때는 <p> 태그를 사용합니다.

pre 태그는
작성한 줄바꿈과 공백을
그대로 화면에 표시합니다.

8. 실습 정리

오늘 실습한 태그는 제목, 문단, 링크, 강조, 인용, 코드 표시 태그입니다.

HTML에서 화면에 보이는 내용은 반드시 body 태그 안에 작성해야 합니다.
