4. overflow 속성

overflow: visible

내용이 박스 높이보다 많아지면 기본적으로 박스 밖으로 그대로 넘쳐서 보입니다. 이것이 visible입니다.

overflow: hidden

내용이 박스 높이보다 많아지면 박스 밖으로 넘친 부분은 잘려서 보이지 않습니다. 이것이 hidden입니다.

overflow: scroll

내용이 많든 적든 항상 스크롤바가 나타납니다. 직접 스크롤해서 숨겨진 내용을 확인할 수 있습니다.

overflow: auto

내용이 넘칠 때에만 자동으로 스크롤바가 나타납니다. 보통 가장 많이 사용하는 방식입니다.
